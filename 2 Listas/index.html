Listas

Ordenadas

Nas listas ordenadas, a ordem importa e os itens são numerados automaticamente. Para criar uma lista ordenada:

  1. Use a tag OL
  2. Escolha o TYPE
    • 1 (Padrão)
    • A
    • a
    • I
    • i
  3. (Opcional: Escolha a primeira numeração com START)
  4. Entre as tags LI, coloque os elementos da lista (O fechamento é opcional)

Não ordenadas

Nas listas não ordenadas, a ordem não importa e todos os itens ficam com o mesmo "marcador". Para criar uma lista Não ordenada:

  1. Use a tag UL
  2. Escolha o tipo de marcador com TYPE
    • disk (padrão)
    • circle
    • square
  3. Entre as tags LI, coloque os elementos da lista (O fechamento é opcional)

ATENÇÃO: Podemos lococar uma lista dentro da outra, sem limite de número e independente da ortenação

Definições

Em uma lista de definição temos o seu termo e sua definição

Iniciamos o codigo com as tags DL
DL = Definition list
Depois colocamos um termo entre tags DT
DT = Definition term
E em seguinda a sua definição entre tags DD
DD = Definition description
