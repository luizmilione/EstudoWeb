Marcações especiais

Nessa aula aprendi a usar as tags EM e STRONG

Podemos ver que <em> faz com que a palavra fique em ênfase
Ja o <strong> faz com que a palavra fique em destaque
Também podemos marcar um temto utilizando <mark>
O texto tambem pode ser considerado deletado usando <del> ou inserido com <ins>.
Para "elevar" alguém, usamos <sup>. Também podemos "rebaixar" usando <sub>

Identação e código fonte

Caso o texto seja um código, podemos usar as tags CODE que deixa o texto assim, todas as letras possuem o mesmo espaço horizontal
Caso dermos enter ao escrever o parágrafo, nada acontecerá na página. Para isso podemos usar a tag BR. Mas se fizermos o texto entre tags PRE, todos os espaços e enters são considerados. ideal para mostrar códigos, por exemplo 

            
    #include stdio.h
    int main()
    printf("Olá mundo em C");

Citações

Para uma frase do tipo citação simples, basta usar a tag Q: Para morrer basta estar vivo
Para uma citação maior ou mais elaborada, usamos BLOCKQUOTE, até mesmo com o suporte para links dentro da tag 

    Eu tava com meus mano lá na minha quebrada
    Chegou o Wanderley e veio dar ideia errada
Podemos usar a tag ABBR para mostrar o significado de uma abreviação. mpo
Com a tag BDO (rtl) o texto pode ficar invertido Ideal para Ambulância
