Uma boa dica para baixar vídeos sem direitos autorais é o Pexels.

É importante usar vários formatos de vídeo pois cada navegador tem uma compatibilidade diferente

  1. O ideal é abrir uma tag VIDEO, adicionar CONTROLS e uma thumb usando POSTER="caminho"
  2. Adicionar varias tags SOURCE:SRC com os vídeos em diferentes formatos
