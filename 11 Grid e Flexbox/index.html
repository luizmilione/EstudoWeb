O que são

São estruturas CSS que trabalham a contensão de elementos baseadoas em posição, tamanho e alinhamento

Layout flex

Indicado para alinhar componentes em uma única dimensão, linha ou coluna.
Os espaçamentos entre os itens são calculados automaticamente

No codigo fica assim:

display: flex;
flex-wrap: wrap;
justify-content: space-between;
align-itens: center;

Grid Layout

É multidimensonal, fazendo com que os elentos tenham interação linha-coluna

É mais complexo, uma vez que é necessário explicitar o fracionamento das divisões (linha/coluna)

O codigo é assim:

display: grid;
grid-templete-columns: repeat(#, #fr); Definimos as colunas e o espaçamento
max-width: ###px; Tamanho máximo da viewport
margin: # auto;
grid-gap: ##px; Espaçamento entre itens

Porém, para cada elemento devemos declarar seus grid-column: #/#; e grid-row: #/#; que indicarão, baseados no grid-templete-columns as posições que irão ocupar
