Sintaxe

No css, dividimos uma declaração em SELETOR { propriedade : valor ; }

Podemos adicionar estilo a todos os elementos de um determinado elemento, como P, H1, ...

Ou podemos atribuir uma id dentro do html para um local específico. Nesse caso, usamos a # para declarar no css

Podemos também fazer uma classe com varios elementos. Nesse caso a declaração no css usa o .

Outro parágrafo da mesma classe

Seletor por atributo, onde todos os elementos com um determinado atributo são selecionados. Ex "link". Na declaração usamos []

Temos tbm as pseudo classes, onde podemos colocar elementos especificos de uma tag , como o primeiro ou aquele cujo mouse está em cima. Na declaração colocamos tag:elemento

Dentro de pseudo classes ainda podemos colocar seletores em pseudo elementos, colocando tag::pseudo

Seletores universais valem para toda a página. Um * antes de {

Combinando seletores

Podemos combinar algumas regras para seletores

  1. A,B: A e B recebem a mesma regra
  2. A.B: Todos os B dentro de A. Similar com o uso de AND &&
  3. A B: Todos os B descendentes de A. Podendo ser qualquer nível de descendencia
  4. A>B: Todos os B filhos de A. Apenas os descendentes de primeiro grau
  5. A+B: B sendo o próximo irmão de A.
  6. A~B: B sendo os próximos irmãos de A

Prioridades

Caso mais de um seletor de mesma especificidade entre em conflito, o último é o que valerá

Caso um seletor seja mais especifico que outro, ele é o que passa a valer

Caso, no CSS, vc coloque !important, essa regra será priorizada
