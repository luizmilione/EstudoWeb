Links externos

Para criar links externos clicáveis, devemos adicionar a tag A (anchor)
Dentro da tag A teremos o href="link do site"

Exemplo: Esse é o vídeo que aprendi isso

Aqui é o buscador mais famoso

É altamente recomendado colocar target="_blank" rel="external". Após issso colocamos o texto clicável e só então fecharemos a tag A. Com isso o site será aberto em outra aba do navegador

Links internos

Em uma página na web, temos que abrir outras páginas dentro do nosso próprio site. Para isso devemos:

  1. Criar outra página .HTML
  2. Abrir a tag A e colocar a referencia da página na pasta
  3. Digitar o texto clicável
  4. Tag A de fechamento
