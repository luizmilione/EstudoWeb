Estilos locais

Para aplicar os estilos localmente (em toda a pagina) devemos criar uma tag STYLE dentro da tag HEAD
Depois criamos seletores para cada tag que é utilizada no html principal

Dessa maneira o seu código fica mais organizado e as alterações são mais fáceis de fazer

Porém existe uma foema ainda melhor de organizar o estilo da sua pagina, o CSS externo
