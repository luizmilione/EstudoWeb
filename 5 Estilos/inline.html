Estilos Inline

Não muito recomendado para fazer o estilo de um site inteiro, o estilo inline é um comando undividual dentro cada tag

Para fazer acontecer, dentro da tag desejada, devemos colocar STYLE = " o que desejar ; "

Essa técnica polui o seu código e aumenta a carga de trabalho ceso seja feito em todo o site. O correto de se usar o Inline é para configurações pontuais
