Propriedades do CSS

Em um arquivo css, ao fazer a referencia para alguma medida, podemos colocar de duas maneiras: Absoluto ou Relativo

Relativo

Os tipos absolutos ocupam um espaço fixo independente da tela, do dispositivo ou qualquer outra coisa

Absoluto

Já os elementos relativos irão ocupar o espaço baseando em algum outro valor a ser definido

Em uma pagina, os elementos podem ser exibidos de maneira inline ou block level

Inline

Os elementos inline são aqueles que não ocupam todo o espaço {horizontal} na pagina, e podem ficar lado a lado com outros elementos

Por padrão, IMG, SPAN, A, BUTTON, INPUT, LABEL, ... são elementos inline

Block level

Os elementos block ocupam todo o espaço na pagina

Por padrão, DIV, Hs, P, FORM, CANVAS, TABLE, SECTION, ... são desse tipo

Inline-block

O display do tipo inline-block é uma especie de mistura dos dois tipos anteriores, os elementos ficam em block, porem inline entre eles

Declaração no CSS

Para mudar o tipo de um elemento, podemos usar DISPLAY: ###;

Podemos usar MARGIN: 0 AUTO; para centralizar um elemento na pagina

Box model

O box model é um conceito fundamental em CSS que descreve como os elementos HTML são renderizados e ocupam espaço em uma página da web. Ele representa cada elemento como uma "caixa" retangular, onde cada caixa tem quatro componentes principais:

  • Conteúdo: Área na caixa com o conteúdo do elemento, como txt, img ...
  • Padding: Área de espaçamento interno, entre o conteúdo e a borda (BORDER)
  • Borda (border): Linha que rodeia a caixa. Pode ser personalizada
  • Margin: Area que envolve a borda. É essencial para controlar o espaçamento entre as diversas caixas na página

Backgrouds

Podemos colocar como backgroud uma imagem. Para isso, devemos utilizar backgroud-image: url(caminho);

Caso queira que a imagem fique grande e no fundo, devemos usar os comandos backgroud-repeat: no-repeat; e também o backgroud-size: 100% 100%; para a imagem ocupar toda a página

Podemos também usar a função linear-gradiente: (cor1, cor2); que fará com que o fundo fique com as cores que variam de cor1 (cima) até cor2

Posicionamentos

A propriedade POSITION organiza os elementos na tela. As 4 principais são

    Static: Considerado como não posicionado, segue o fluxo de renderização da página
  • Relative: Segue o fluxo de renderização da pagina, porém é afetado pelas Propriedades TOP, BOTTON, LEFT, RIGHT e Z-INDEX
  • Fixed: Não segue o fluxo de renderização, ficando estático e fixo na tela
  • Absolute: Não segue o fluxo de renderização, ficando ligado ao seu ancestral mais próximo. pode ser ajustado usando as propriendades TOP, ... . Não é reposicionado

Por padrão, o podicionamento dos elementos são sempre static

As Propriedades TOP, BOTTON, LEFT e RIGHT posicionam o elemento a partir de sua respectiva borda, ou seja, top: 20px; move o elemento para baixo

A propriedade Z-INDEX move o elemento no indice Z, ou seja, caso ocorra um conflito entre 2 elementos, aquele com o z-index maior irá sobrepor
